HTML Fundamental tags

Tag Name Description
<!DOCTYPE html> Document Type This declaration tells the browser the type of document
and the HTML version to expect
<html> HTML tag Anything between the opening and closing of the html
tag, will be interpreted as html code
<Head> The head This contains the metadata of the webpage
<Title> Page Title This is within the head section and it contains the title
of the web page
<body> The Body This contains all the elementes that are going to be visible on the webpage
<h1> H1 tag This contains the title(heading) on the webgape (h1,h2,h3...h6)
<p> paragraph This tag is used for plain text
<div> divisionh This tag creates a division between other elements in the webpage.
<a> anchor element This is used to add a link to a webpage
<!---here---> comment This allows us to add comment to code

HTML table tags

Tag Name Description
<table> table The wrapper element for all HTML tables.
<thead> table head The set of rows defining the column headers in a table.
<tbody> table body This should include all data other than the table header
<tr> table row The table row container, adds rows to the table.
<td> table data This add data to the cell.
<th> table heading This is used to add titles to rows and column.
<tfoot> table foot This is used to section out the bottom part of a table
<th scope="row"> table heading row This add a table header for row.
<th scope="col"> table heading column This add a table header for column
<td colspan> td colspan="2" This spans 2 columns
<td rowspan> td rowspan="3" This spans 3 rows
